I'm a final-year CS/AI student at HKUST passionate about **multimodal AI that works reliably in the real world** — especially where data is noisy, incomplete, or life-critical.

My current focus:
- Fall detection & elderly monitoring using mmWave radar point clouds + Point4D transformers (ongoing thesis w/ Prof. Gary Chan)
- Multimodal foundation models for cancer diagnosis/prognosis (w/ Prof. Hao Chen): built annotation pipelines, scaled InternVL-8B on 4×A40, integrated 500GB+ AFNI neuroimaging data
- LLM data quality engineering: evolved instruction tuning + custom cleaning pipelines to cut training noise (earlier w/ Prof. Xiaofang Zhou)

Previously interned at iFlytek Consumer Big Data, researched KG+LLM at USTC FuSEP, and TA'd intro CS. TOEFL 103 / GRE 323+.

**Open to** → 2026 full-time AI Researcher / ML Engineer roles (healthcare AI, multimodal, edge sensing) or top PhD programs. Always happy to chat about mmWave + transformers, data-centric AI, or medical imaging.
